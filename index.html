Язык

Введение:
Хочешь выучить новый язык, но не хочешь учить латино-подобный язык?
Мы разрабатываем язык, основанный на русском и правилах эсперанто.

Для начала изучим алфавит и значение каждой буквы в этом языке...
В самом начале нас встречает буква А, которая при добавлении в конец слова означает существительное.
Следом идёт буква Б, которая ничего не означает в слове, но используется для корня слова.
Далее есть буква Б̃, которая собой представляет две буквы - Б и Ь, используется для сокращения русского письма.
За предидущей буквой идёт буква Б̅, которая означает две буквы - Б и Ъ, используется для сокращения русского письма.
